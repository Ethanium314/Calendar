# Calendar

## Day view:

- arrow keys: movement
- ctrl-S: Save (Not yet implemented)
- ctrl-Q: Quit
- escape: return to month view
- shift-right: next day
- shift-left: previous day

## Month view:

- arrow keys: movement
- ctrl-S: Save (Not yet implemented)
- ctrl-Q: Quit
- enter: enter day view of selected day
- shift-right: next month
- shift-left: previous month
